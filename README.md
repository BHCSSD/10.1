# 10.1
## 10.1-Common-Errors
1. **Basic Print Statements**:
   - **print() Statements**: Understanding how `print()` and `text()` functions display output.
   - **String Concatenation**: Using `+` to concatenate (join) strings in print statements.

2. **Recognizing Common Errors**:
   - **Syntax Errors**:
     - Forgetting semicolons.
     - Unclosed quotes or brackets.
   - **Case Sensitivity**:
     - Mis-capitalization issues.
   - **Bracket Mismatches**:
     - Adding one too many closing brackets.
     - Removing a necessary closing bracket.

3. **Function Placement**:
   - **Moving text() outside of draw**: Understanding where certain functions should be placed within the code.

4. **p5.js Drawing Functions**:
   - **Basic Shapes**:
     - `circle()`, `rect()`, and `ellipse()` functions.
   - **Style Functions**:
     - `stroke()`, `strokeWeight()`, and `fill()` functions.
   - **Canvas Setup**:
     - Using `createCanvas()` and setting up the background.

6. **Code Structure**:
   - **Functions in p5.js**:
     - Understanding the structure and roles of `setup()` and `draw()` functions.
   - **Commenting**:
     - Using comments to document code (`//setup`, `//draw`, `// camelCase`).
